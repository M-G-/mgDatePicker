mgDatePicker jQuery时间选择插件 DEMO

默认： <input type="text" data-datepicker value="2016-07-01"/>

位置偏移： <input type="text" data-datepicker="top:-33,left:254"/>

限制时间范围： <input type="text" data-datepicker="from:2016-6-4,to:now"/>

选择时间： <input type="text" data-datepicker="type:time"/>

选择日期和时间： <input type="text" data-datepicker="type:date-time"/>
